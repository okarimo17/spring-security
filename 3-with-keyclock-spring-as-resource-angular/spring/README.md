# Spring Boot Resource Server (with Angular Frontend)

This project is the backend part of a full-stack example. It is a Spring Boot application acting as a resource server, secured by Keycloak. It works together with the Angular frontend in the `front` folder.

**Key features:**
- Spring Boot and Spring Security
- Resource server configuration
- Secured endpoints using Keycloak
- Designed to work with the Angular frontend for a complete authentication flow

Use this to learn how to secure both backend and frontend with Keycloak.
